###Installation

* Install **conky**.
    * If you use **numantinos** or **arch**, try: <code>$sudo pacman -S conky</code>

* **Optional** Install the [Poiret One](https://www.google.com/fonts/specimen/Poiret+One) font.

* Use the '.install.sh' script
